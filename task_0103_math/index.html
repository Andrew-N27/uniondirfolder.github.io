Интегралы основные формулы

# $${Формула}$$
1 $${∫ (0) dx=C}$$
2 $${∫ (a) dx = ax + C (a = const)}$$
3 $${∫ {x^n} dx = {x^{n+1} \over n + 1} + C (n \ne -1)}$$
4 $${∫ {dx \over x} = ln |x| + C}$$
5 $${∫ {a^x} dx = {a^x \over ln (a)} + C}$$
6 $${∫ e^x dx = e^x + C}$$
7 $$ ∫ { {dx} \over {\sqrt {x^2 \pm a^2}}} = ln |x + \sqrt {x^2 \pm a^2|}$$
